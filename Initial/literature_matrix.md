# Literature Matrix: Adaptive Uncertainty Labeling for Imbalanced Multi-label CXR Classification

> 推荐 topic：**Adaptive Uncertainty Labeling for Imbalanced Multi-label Chest X-ray Classification**  
> 中文：**面向类别不平衡多标签胸片分类的自适应不确定标签处理方法研究**  
> 说明：根据 Chapter 2 Literature Review 课件要求，本矩阵不把文献做成简单 annotated bibliography，而是围绕数据标签不确定性、长尾多标签分类、label-wise analysis 和医学影像可靠性进行 synthesis。

## Matrix-level Critical Synthesis

CheXpert 原论文已经系统比较过几种 uncertain label handling strategies，而且它并不是完全使用单一固定策略：论文明确发现不同 pathologies 适合不同策略，并在最终模型中为 5 个 competition tasks 选择了不同 uncertainty handling approaches。因此，本 FYP **不能**把 gap 写成“uncertain labels 尚未被研究”，也不能简单写成“现有研究没有 label-wise uncertainty handling”。

真正可行的 gap 是更细的：

> CheXpert 已经做了 selected pathologies 的 per-label strategy comparison，但其核心分析集中在 5 个 competition labels，并主要使用 AUROC。对于 imbalanced multi-label setting 中不同 prevalence 和 uncertainty burden 的标签，uncertain label handling 的影响仍缺乏系统研究，尤其缺少同时基于 label prevalence、uncertain ratio 和 uncertain/positive ratio 的 group-wise analysis。

因此，本 FYP 的贡献不应写成“首次做 label-wise uncertainty handling”，而应写成：

> 在 CheXpert 既有 uncertainty strategy 基础上，进一步研究 label imbalance 和 uncertainty burden 如何影响 uncertain label handling，并提出 teacher-guided label-wise adaptive uncertainty soft labeling：结合 label-level uncertainty burden prior 和 teacher model 的 instance-level prediction，为每个 uncertain label 生成不同 soft target，再用 AUPRC、F1、Recall 等 imbalance-sensitive metrics 进行系统评价。

该方向与 CheXpert 的 U-SelfTrained 有联系，但差异在于：CheXpert 用模型预测重标 uncertain labels，并在 5 个 competition tasks 上比较；本项目则显式加入 label-level uncertainty burden 和 label imbalance analysis，重点评价 rare labels 与 high-uncertainty-burden labels，而不是只追求 selected common labels 的 AUROC。
